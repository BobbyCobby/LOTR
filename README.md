# LOTR

This is eventually going to be a base ruleset LOTR mod. At time of writing, this file is all that's here!

My first step will be a large map of Middle-Earth. I doubt it's legal to distribute the reference image, and there's no point--it's available at [https://tolkiengateway.net/wiki/File:Christopher_Tolkien_-_The_West_of_Middle-earth.png](https://tolkiengateway.net/wiki/File:Christopher_Tolkien_-_The_West_of_Middle-earth.png).

I welcome any contributions! Please submit them as a pull request. There's loads of information on the Internet about how; I'd recommend the GitHub docs or just looking it up. I'm happy to help, although I don't claim to be a master of Git. My main recommendation would be to use GitHub Desktop; I've done a lot of modding (mostly private) and barely ever need Git on the command line, although I'm sure it's more powerful.

I also think it'd be fun to make a mod of this mod (when there's something to mod) that translates everything in Unciv into Sindarin or Quenya. If there is anyone reading this who's better at Elvish than I am, which isn't that great, I'd love to work together on that!

I'll make this more organized when the mod exists as more than a README.
